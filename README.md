# Frontend

## Overview

이 프로젝트는 [React](https://reactjs.org/)를 사용하여 개발된 프론트엔드 애플리케이션입니다.

**허용된 IP 접근 제한 설정 프로젝트**의 사용자 인터페이스와 클라이언트 측 로직을 담당합니다.


## Prerequisites

- Node.js >= 18.15.0
- Yarn >= 1.22.19


## Installation

   ```bash
   git clone https://github.com/parfait2/flow-front.git

   cd your-repo/frontend
   
   yarn install
   
   yarn start
